Credit Card Fraud Detection

Problem Statement:¶

With the growth of e-commerce websites, people and financial companies rely on online services to carry out their transactions that have led to an exponential increase in the credit card frauds. The design of an effective fraud detection system is necessary in order to reduce the losses incurred by the customers and financial companies. The Credit Card Fraud Detection Problem includes modeling past credit card transactions with the knowledge of the ones that turned out to be fraud. This model is then used to identify whether a new transaction is fraudulent or not. A good fraud detection system should be able to identify the fraud transaction accurately and should make the detection possible in real-time transactions. Mis-used (Supervised Learning) and Anomaly detection (Unsupervised Learning) systems bring normal transaction to be trained and use techniques to determine novel frauds. From a set of given data we have to design the machine learning algorithm which best fits the data and able to recognise the fraud transaction accurately.¶

6. Conclusion¶

Supervised Learning Algorithms¶

From the above table we can see that both Support Vector Machine and Logistic Regression same accuracy where K-Nearest Neighbor having highest efficiency. The difference in efficiency is also small so we need to check other metrics. All of supervised learning algorithms are varring in Precision and Recall. In our case, predicting the valid transaction which is actually a fraud (False Negative) can not be tolerate. So our algorithm should have minimum False Negative cases. Similarly predicting the fraud transaction which is actually a fraud (True Positive) is more important than predicting fraud transaction for valid transaction (False Positive). Although we want our algorithm should give minimum False Positive. High Precision implies low False Positive rate and high Recall implies low False Negative rate. As per our conditions our algorithm should have low False Negative rate i.e High Recall.

Accuracy score, Recall, Precision, F1 score indicating that K-Nearest Neighbors is the best algorithm (of all tested) for predicting credit card fraud trasaction in case of supervised learning.¶

Unsupervised Learning Algorithms¶

In case of unsupervised learning Isolation Forest has higher accuracy, Precision and Recall. So we can say that Isolation Forest is the goad algorithm for our task.

Overall observations concludes that K-Nearest Neighbors is the best algorithm (of all tested) for detecting the credit card fraud transactions for the given data.¶

If computation time/cost is a concern in training and testing the algorithm then we might go with Support Vector Machine. As per the results it is the second most efficient algorithm after K-Nearest Neighbors.
